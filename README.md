# 点子开关知识库助手 GitHub Pages Demo

这是一个用于发布 Dify 聊天助手的 GitHub Pages 静态页面。

## 使用方法

1. 在 GitHub 新建一个仓库，例如 `dify-chat-demo`。
2. 上传本文件夹中的 `index.html`。
3. 打开仓库的 `Settings`。
4. 进入 `Pages`。
5. 在 `Build and deployment` 中选择 `Deploy from a branch`。
6. 分支选择 `main`，目录选择 `/root`。
7. 保存后等待 GitHub Pages 生成访问链接。
